# Hot Wheels Original 16

A retro-styled landing page celebrating the original Hot Wheels die-cast cars from 1968.

## Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Features

- Responsive mobile-first design
- Catalog with expandable card grid
- Email signup modal
- Smooth scroll animations
- Mobile hamburger menu

## Run

Open `index.html` in any browser.

## Structure

```
├── index.html      # Main page
├── styles.css      # Custom styles
├── script.js       # Interactions
├── 1-8.png         # Car images
├── car.png         # Hero car
└── road-texture.jpg
```
